HOID Project

HOID Project is an Indonesian History Mod Project based on the World conqueror 4 (WC4) Game by EasyTech

This Project is a rework of the base game with the plan such adding Indonesian Scenarios, Indonesian Historical Battle & more.

Currently This Project is Developed by HOID Team.
